---
title: 용어집
description: OpenClaw 관련 주요 용어 정리
---

## A

**Agent (에이전트)**: OpenClaw의 AI 코어. 사용자 요청을 이해하고 실행합니다.

**API Key**: AI 모델 서비스(OpenAI, Anthropic 등) 접근을 위한 인증 키.

## C

**Channel (채널)**: OpenClaw와 연결된 메신저 플랫폼(Telegram, Slack 등).

**ClawHub**: OpenClaw 스킬 마켓플레이스.

## D

**Daemon (데몬)**: 백그라운드에서 실행되는 OpenClaw 서비스.

**DM**: Direct Message. 비공개 메시지.

## G

**Gateway**: OpenClaw의 중앙 제어 서버. WebSocket 기반.

## M

**Model (모델)**: AI 언어 모델(GPT-4, Claude 등).

## P

**Profile (프로필)**: 독립된 OpenClaw 설정 환경.

**Prompt Injection**: 악성 입력을 통한 AI 조작 공격.

## S

**Skill (스킬)**: OpenClaw의 확장 기능 플러그인.

**Sandbox**: 격리된 실행 환경.

## W

**Workspace (워크스페이스)**: 프로젝트별 작업 공간.

**WSL2**: Windows Subsystem for Linux 2.

---

## 관련 용어

- **LLM**: Large Language Model (대형 언어 모델)
- **MCP**: Model Context Protocol
- **Webhook**: HTTP 콜백 이벤트
- **Cron**: 작업 스케줄러
