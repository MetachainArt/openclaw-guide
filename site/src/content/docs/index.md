## OpenClaw란?

OpenClaw(구 Moltbot/Clawdbot)는 사용자의 컴퓨터에서 **로컬로 실행**되는 자율 AI 에이전트입니다. 텔레그램, 슬랙, WhatsApp 등 메신저와 연결하여 다양한 자동화 작업을 수행합니다.

### 주요 특징

- 🔒 **로컬 실행**: 모든 데이터가 내 컴퓨터에 저장
- 🔗 **멀티 채널**: WhatsApp, Telegram, Slack, Discord, iMessage 등 50+ 연동
- 🤖 **자율 실행**: 크론 작업, 이벤트 기반 자동화
- 🧩 **스킬 확장**: 3,500+ 커뮤니티 스킬
- 🌐 **브라우저 제어**: Playwright 기반 웹 자동화
- 🎤 **음성 지원**: 음성 명령 및 응답

## 누구를 위한 책인가?

| 대상 | 추천 챕터 |
|------|----------|
| **초급자** | 빠른 시작 → 설치 가이드 → 초급 활용 사례 |
| **재입사자/이전 사용자** | 설치 가이드(변경사항) → 운영 가이드 |
| **중급 개발자** | 활용 사례(중급) → 팀 에이전트 운영 → 보안 가이드 |

## 최신 정보 (2026년 2월 기준)

### 시스템 요구사항
- **Node.js**: 22+ (필수)
- **OS**: macOS, Linux, Windows (WSL2 권장)
- **메모리**: 최소 4GB 권장
- **디스크**: 2GB 이상 여유 공간

### 권장 설치 방법 (공식)

```bash
# macOS / Linux / WSL2
curl -fsSL https://openclaw.ai/install.sh | bash

# Windows (PowerShell) - WSL2 권장
iwr -useb https://openclaw.ai/install.ps1 | iex
```

온볼딩 마법사 실행:
```bash
openclaw onboard --install-daemon
```

:::danger[⚠️ ClawHub 보안 사고 (2026년 2월)]
