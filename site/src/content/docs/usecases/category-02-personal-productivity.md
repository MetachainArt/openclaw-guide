<h2 id="case-1">1. 두 번째 뇌(Second Brain) 구축</h2>

### 📖 상세 설명

번뜩이는 아이디어나 링크, 메모를 채팅으로 별낸 다음, 자동으로 맥락과 함께 분류하여 저장합니다. 나중에 검색할 때 관련성 높은 과거 기록도 함께 찾아주어 개인 지식베이스를 구축합니다.

**어떻게 작동하나요?**
1. 메신저로 아이디어/링크/메모 전송
2. AI가 내용 분석 및 자동 분류
3. 벡터 임베딩으로 의미 기반 저장
4. 검색 시 관련성 높은 정보 함께 제공

### 🛠️ 필요 스킬

| 스킬 | 용도 | 설치 명령어 |
|------|------|------------|
| memory | 메모 저장 | `openclaw skill install memory` |
| notes | 노트 관리 | `openclaw skill install notes` |
| search | 벡터 검색 | `openclaw skill install search` |
| embedding | 의미 기반 임베딩 | `openclaw skill install embedding` |

### 📝 단계별 설정

**Step 1: Obsidian 연동**

```bash
openclaw skill install memory notes
# Obsidian 볼트 경로 설정
```

**Step 2: 자동 분류 설정**

```json
{
  "memory": {
    "vault": "~/Documents/Obsidian/SecondBrain",
    "autoTag": true,
    "dailyNotes": true,
    "categories": {
      "아이디어": ["idea", "thought", "아이디어"],
      "리소스": ["link", "article", "resource"],
      "프로젝트": ["project", "task", "할일"],
      "학습": ["learn", "study", "공부"]
    }
  }
}
```

### 💬 사용 예시

**메모 저장:**
```
User: 아이디어: AI로 이력서 자동 분석해서 합격률 예측하는 서비스

OpenClaw:
🧠 메모 저장 완료

📁 저장 위치: Inbox/아이디어.md
🏷️ 자동 태그: #아이디어 #AI #서비스 #이력서
🔗 유사 메모: 
   - "AI 면접 코치 앱 아이디어" (85% 유사)
   - "구직 매칭 플랫폼 구상" (72% 유사)

💡 다음 행동 제안:
   1. 시장조사 템플릿 열기
   2. 관련 기사 검색
   3. 경쟁사 분석

[📝 메모 보기] [🔍 유사 아이디어] [📊 시장조사 시작]
```

**검색:**
```
User: 이력서 관련 아이디어 찾아줘

OpenClaw:
🔍 검색 결과: 3개

1. 💡 AI 이력서 분석 서비스 (2시간 전)
   "AI로 이력서 자동 분석해서 합격률 예측..."
   관련성: 95%

2. 💡 AI 면접 코치 (1주일 전)
   "AI가 면접 질문 예측하고 모의 면접..."
   관련성: 85%

3. 💡 구직 매칭 플랫폼 (2주일 전)
   "이력서 키워드 매칭으로 맞춤 채용공고..."
   관련성: 72%

📊 연결 제안:
이 3개 아이디어를 통합하면
"AI 기반 취업 플랫폼"이 됩니다.

[🔗 아이디어 연결하기] [📋 비즈니스 모델 검토]
```

<h2 id="case-2">2. 음성 메모 → 일기 자동 변환</h2>

### 📖 상세 설명

길 가다가 남긴 30초짜리 짧은 음성 메모를 AI가 전사하여 "오늘의 하이라이트" 형태로 멋진 일기로 정리합니다. 음성 톤에서 감정을 분석하고, 관련 사진을 자동으로 찾아 삽입합니다.

### 💬 사용 예시

```
User: (음성 메모 30초)
"오늘 회사에서 프레젠테이션 잘 마쳤어. 
팀장님이 칭찬하셨고, 클라이언트도 만족한 것 같아."

OpenClaw:
📔 오늘의 일기 (2월 25일 화요일)

🎯 하이라이트:
프레젠테이션 성공적으로 마무리! 
팀장님의 칭찬과 클라이언트의 긍정적 반응을 얻었다.
업무 성취감이 큰 하루.

😊 감정 분석: 긍정적 (85%)
   자신감, 성취감, 만족감

📸 오늘의 사진:
   (프레젠테이션 장면, 점심 사진 자동 연결)

✨ 한 줄 평:
"성취의 맛을 본 화요일"

📁 Obsidian에 저장 완료
[📖 일기장 보기] [📊 감정 트렌드] [🖼️ 사진 추가]
```

<h2 id="case-4">4. 집중 모드 가디언 (Focus Mode Guardian)</h2>

### 📖 상세 설명

딥워크(Deep work)를 할 때는 알람을 차단하고, 정말 긴급한 연락만 선별해서 화면에 표시합니다. 집중력을 높이면서도 중요한 일은 놓치지 않습니다.

### 💬 사용 예시

```
User: 집중 모드 시작, 2시간 동안

OpenClaw:
🎯 집중 모드 활성화 (2시간)

🔕 차단됨:
• 일반 알림 (Slack, 이메일 등)
• 뉴스/소셜 미디어
• 비긴긴 연락

✅ 허용됨:
• 📞 전화 (2회 이상 연속)
• 🚨 긴급 메일 (보안, 장애)
• ⏰ 집중 시간 종료 알림

📊 집중 모드 통계:
• 이번 주 집중 시간: 12시간
• 평균 집중도: 85%
• 방해 요소 차단: 23개

🎵 추천 집중 음악 재생중...

[⚠️ 긴급 상황 발생 시 '긴급'이라고 별내주세요]
```

<h2 id="case-5">5. 작업 시간 자동 추적 (Time Tracking)</h2>

### 📖 상세 설명

어떤 앱을 썼고 누구와 대화했는지 분석하여 프로젝트별 소요 시간을 자동으로 기록합니다. 수동으로 시작/정지할 필요 없이 AI가 자동으로 인식합니다.

### 💬 사용 예시

```
User: 오늘 작업 리포트 보여줘

OpenClaw:
📊 2월 25일 작업 리포트

⏰ 총 작업 시간: 7시간 23분

📂 프로젝트별 시간:
• 프로젝트 A (클라이언트 X): 3시간 15분
  - VS Code: 2시간
  - Slack (#proj-a): 1시간 15분
  
• 프로젝트 B (내부): 2시간 45분
  - Figma: 1시간 30분
  - 회의: 1시간 15분
  
• 행정/기타: 1시간 23분
  - 이메일: 45분
  - 문서 작업: 38분

⚠️ 주의: 소셜 미디어 45분 소비

💡 제안:
내일은 프로젝트 A에 4시간 집중하세요.
마감이 3일 남았습니다.

[📊 주간 리포트] [⏱️ 클라이언트 과금서 생성]
```

```
[아빠] 님이 장을 보러 가는 중입니다.

추가로 필요한 것 있으세요?
```
