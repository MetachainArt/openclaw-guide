<h2>1. 멀티플랫폼 카피 생성</h2>

```
User: 신제품 "스마트워치" 카피 생성
특징: 건강모니터링, 7일 배터리

OpenClaw:
📝 멀티플랫폼 카피 완료

🐦 X (40자):
건강을 손목에서. 7일 배터리로 끊김없이.
#스마트워치

📸 Instagram:
당신의 건강 파트너 ⌚
24시간 건강모니터링 + 7일 배터리
더 이상 충전 걱정 없이
링크 in bio 👆

💼 LinkedIn:
직장인의 건강 관리 혁신
AI 기반 건강모니터링 스마트워치 출시
7일 배터리로 실용성 극대화

📧 이메일 (제목):
[30% 할인] 당신의 건강을 지켜줄 스마트워치

[📋 전체 보기] [📤 바로 게시]
```

<h2>2. 브랜드 멘션 모니터링</h2>

```
OpenClaw (실시간):
🔴 긴급: 부정적 멘션 급증

키워드: @mycompany
감성: 매우 부정적 (0.85)

💬 원문:
"@mycompany 이번 업데이트 개판이네요"

📊 상황 분석:
- 리트윗: 23회 (10분 내)
- 유사 멘턴: 5건 추가

⚠️ 위험도: HIGH
[빠른 답변 보내기] [상세 분석 보기]
```

<h2>3. 광고 A/B 테스트 자동화</h2>

```
User: 페이스북 광고 A/B 테스트 시작
변수: 헤드라인 2개

OpenClaw:
🧪 A/B 테스트 설정 완료

변수 A: "지금 바로 시작하세요"
변수 B: "7일 무료로 경험해보세요"

📊 예상:
• 테스트 기간: 7일
• 예산: ₩100,000/버전
• 유의미 차이: CTR ±15%

🔄 자동 최적화:
48시간 후 우수 버전으로 80% 예산 배분

[🚀 테스트 시작] [⚙️ 설정 변경]
```
