<h2 id="case-1">1. 영수증 사진 → 엑셀 자동 입력</h2>

### 💬 사용 예시

```
User: (영수증 사진 첨부)

OpenClaw:
📸 영수증 분석 완료

💰 추출 정보:
- 상호: 스타벅스 강남점
- 날짜: 2026-02-25
- 금액: ₩8,500
- 품목: 카페라떼(Tall)

🏷️ 카테고리: 식비/커피 (자동 분류)

✅ Google Sheets 저장 완료
   행 47에 추가됨

📊 이번 달 식비 현황:
   총 지출: ₩127,500
   예산 대비: 85% 사용
   ⚠️ 예산 초과 예상 (3일 남음)

[📈 상세 보기] [💰 예산 조정] [📑 세무 리포트]
```

<h2 id="case-3">3. 데이터베이스 요약 리포트</h2>

### 💬 사용 예시

```
User: 이번 달 매출 현황 알려줘

OpenClaw:
📊 2월 매출 리포트

💰 총 매출: ₩425,000,000
   전월 대비: +12.5%
   전년 동월 대비: +28.3%

📈 일별 추이:
최고: 2/14 (₩21M) - 발렌타인 이벤트
최저: 2/3 (₩8M) - 설연휴
평균: ₩15.2M/일

🏆 TOP 3 제품:
1. 프리미엄 패키지 (₩85M)
2. 스타터 키트 (₩62M)
3. 엔터프라이즈 (₩58M)

👥 고객 현황:
• 신규: 234명
• 재구매: 567명
• 이탈률: 2.3%

💡 AI 인사이트:
• 주말 매출이 평일보다 35% 높음
• 모바일 결제 비중 67%로 증가
• 신규 고객의 2차 구매율 상승 중

📧 상세 리포트 이메일로 발송 완료
[📊 대시보드 보기] [📥 CSV 다운로드] [🔄 실시간 모니터링]
```
