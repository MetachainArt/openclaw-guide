<h2>1. 사업 아이디어 검증</h2>

```
User: "AI 이력서 검토 서비스" 사업성 검토해줘

OpenClaw:
💼 사업성 분석 리포트

📊 시장 분석
• TAM: ₩500B (글로벌 HR테크)
• SAM: ₩50B (국내 채용시장)
• SOM: ₩5B ( achievable)

🏆 경쟁 분석
강점: 기술 차별화 가능
약점: 기존 플레이어 브랜드 인지도
기회: 원격 채용 증가 추세
위협: 대기업 진입 가능성

💰 수익 모델
• 구독: ₩50,000/월
• 과금: ₩500/건
BEP: 월 500구독자

⚠️ 리스크
• 데이터 프라이버스
• AI 편향성 이슈
• 규제 리스크

💡 추천: 실험적 진행 (MVP 개발)
```

<h2>3. KPI 대시보드 & 알림</h2>

```
User: 오늘 CEO 브리핑 준비해줘

OpenClaw:
📊 CEO 모닝 브리핑 (2월 26일)

💰 매출 현황
• 이번 달: ₩850M (목표 92%)
• 전월 대비: +15%

👥 인사
• 신규 입사자: 3명
• 퇴사 예정: 1명

🚀 프로젝트
• 진행중: 8개
• 지연: 1개 (프로젝트 X, 2일)

⚠️ 이슈
1. 서버 비용 초과 (15%)
2. 클라이언트 A 컴플레인

💡 AI 제안
• 퇴사률 상승 추세 → 1:1 면담 검토
• 프로젝트 X → 리소스 추가 투입

[📊 상세 대시보드] [📧 공유]
```
